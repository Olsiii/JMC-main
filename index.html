Preview the JMC website

To see the actual site instead of a file list:

  1. Open a terminal in JMC-main/frontend.
  2. Run yarn start (or npm start).
  3. Open http://localhost:3000 in your browser.

For a production-style preview, run yarn build then npx serve -s build -l 3000 in the frontend folder, then open http://localhost:3000.
